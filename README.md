# Python Machine

Simulate Cell Machine inside python

## Instalation

```bash
$ pip install 
```

## Usage

```python
from Python_Machine import CellMachine

cellmachine = CellMachine()
cellmachine.parse_code("...")

...Doing actions on the level...

cellmachine.view().show()
```
